# Early-Trend-Analyser
Early Trend Analyser : EarlyTrend AI is an AI-powered social media trend analyzer that detects emerging trends before they go viral. It helps creators and businesses discover trending topics, hashtags, and content opportunities in real time.


# EarlyTrend AI

EarlyTrend AI is an MVP blueprint and starter scaffold for an early social media trend detection SaaS platform. It combines trend scoring, multi-platform intelligence, and creator opportunity recommendations.

## Project structure
- `project/backend` - FastAPI backend scaffold
- `project/frontend` - React frontend scaffold
- `project/docs` - architecture and product documentation

 Getting started

Backend
1. Navigate to `project/backend`
2. Create a `.env` file from `env.example` if available.
3. Install dependencies and run the app with Uvicorn.

# Frontend
1. Navigate to `project/frontend`
2. Install packages: `npm install`
3. Start dev server: `npm run dev`

# Tech stack
- Frontend: React, Vite, React Router
- Backend: FastAPI, Pydantic, JWT
- Database: PostgreSQL (recommended)
- AI/ML: Python libraries like scikit-learn and sentence-transformers
